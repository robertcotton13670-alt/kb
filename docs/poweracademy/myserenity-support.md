# Support technique

Trois moyens de contacter le support technique Poweriti pour créer un ticket d'intervention.

## Option 1 — Icône Poweriti (recommandée)

1. Repérer l'icône Poweriti (girofar ou cadenas) dans la barre des tâches système
2. Faire un clic droit
3. Sélectionner "Demande de support" ou "Capture d'écran avec demande de support"

!!! tip "Capture d'écran automatique"
    L'option avec capture d'écran joint automatiquement une image de votre écran au ticket — ce qui accélère le diagnostic.

## Option 2 — E-mail

Envoyer un e-mail à **help@robaspowerit.com**  
→ Un ticket est créé automatiquement dans le système Poweriti.

## Option 3 — Téléphone

Appeler le **04-120-40-190** → touche 1 pour le support technique informatique.

## Informations utiles pour un ticket efficace

- Décrire précisément le problème rencontré
- Indiquer depuis quand le problème est présent
- Préciser si d'autres collègues sont impactés
- Joindre une capture d'écran si possible
