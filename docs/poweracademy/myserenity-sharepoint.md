# SharePoint

SharePoint remplace le serveur de fichiers local (lecteur réseau) par un espace cloud collaboratif. Les répertoires partagés de l'organisation y sont hébergés et accessibles depuis n'importe quel appareil.

## Accès à SharePoint

| Support | Accès |
|---|---|
| PC | Explorateur de fichiers — icône immeuble avec le nom de l'organisation |
| Smartphone / Tablette | Application mobile OneDrive |
| Navigateur | Portail Microsoft 365 → SharePoint |

!!! tip "Deux icônes dans l'explorateur"
    Le nuage bleu = OneDrive personnel. L'icône immeuble = sites SharePoint partagés de l'entreprise.

## Organisation des répertoires

Les répertoires peuvent être organisés par thématique : comptabilité, commerce, direction, commun...  
Chaque site peut être personnalisé visuellement avec des boutons, raccourcis et une base de connaissance interne.

!!! warning "Administration réservée"
    La création et l'administration d'un site SharePoint sont réservées à un administrateur habilité.

## Bonnes pratiques

- Stocker les fichiers collaboratifs dans SharePoint, pas en local
- Partager les fichiers via un lien Teams plutôt qu'en pièce jointe e-mail
- Ne jamais stocker de documents importants dans le dossier Téléchargements (non sauvegardé)
