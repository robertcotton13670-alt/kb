# Antispam VADE

VADE est une solution antispam française (société lilloise), leader mondial de la défense prédictive de messagerie. Elle protège 1,5 milliard de boîtes mail dans 76 pays et est intégrée au cœur de Microsoft 365.

## Ce que VADE fait automatiquement

- Filtre les e-mails malveillants avant qu'ils n'atteignent la boîte de réception
- Analyse les liens (environnement sandbox isolé) et les pièces jointes
- Classe automatiquement les e-mails publicitaires dans un dossier dédié
- Protège contre phishing, spear phishing, malware et ransomware

!!! tip "Transparent pour l'utilisateur"
    VADE fonctionne en arrière-plan. Aucune action n'est requise au quotidien.

## Types de menaces couvertes

| Menace | Description |
|---|---|
| Phishing | Usurpation d'un organisme connu pour voler des identifiants via un faux lien |
| Spear phishing | Phishing ciblé usurpant l'adresse d'un interlocuteur connu |
| Malware | Logiciel malveillant transmis par pièce jointe ou lien |
| Ransomware | Malware chiffrant les fichiers et exigeant une rançon |

## Limites — votre vigilance reste indispensable

!!! warning "VADE ne remplace pas la vigilance"
    Aucun filtre n'est infaillible. En cas de doute sur un e-mail reçu :

    - Ne pas cliquer sur les liens
    - Vérifier l'adresse exacte de l'expéditeur
    - Appeler l'expéditeur par téléphone pour confirmer
