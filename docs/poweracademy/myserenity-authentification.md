# Authentification & MFA

L'accès aux outils MySerenity repose sur un identifiant unique (adresse e-mail professionnelle) associé à un mot de passe fort et à la double authentification (MFA).

## Se connecter à Microsoft 365

1. Ouvrir un navigateur en navigation privée
2. Aller sur [www.office.com](https://www.office.com) ou rechercher "Login Microsoft 365"
3. Cliquer sur "Connexion"
4. Saisir l'adresse e-mail, puis le mot de passe
5. Valider la double authentification (SMS ou Microsoft Authenticator)
6. Cliquer sur le bloc de 9 points (en haut à gauche) pour accéder aux applications

!!! warning "Navigation privée obligatoire"
    Sur un poste partagé ou invité, utiliser impérativement la navigation privée pour ne laisser aucun identifiant en mémoire.

## Mot de passe fort

Un mot de passe fort comprend au minimum 12 caractères incluant majuscules, minuscules, chiffres et caractères spéciaux.

**Méthode recommandée :** choisir une phrase mémorable + date + caractère spécial.  
Exemple : `Soleil2024!` → `MonSoleilBrille2024!`

## Double authentification (MFA)

Le MFA est imposé par Microsoft pour toute connexion depuis un nouveau périphérique ou navigateur. Deux méthodes disponibles :

- Code SMS envoyé sur le téléphone enregistré
- Validation via l'application Microsoft Authenticator

!!! tip "Recommandation"
    Activer le MFA sur tous les services personnels qui le permettent (banque, messagerie personnelle, réseaux sociaux).
