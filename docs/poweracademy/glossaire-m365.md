# Glossaire Microsoft 365

## Exchange

Service de messagerie professionnelle Microsoft hébergé dans le cloud. Synchronise e-mails, agendas, contacts et tâches sur tous les périphériques en temps réel.

## Outlook

Client Microsoft pour gérer e-mails, agendas, contacts et tâches. Accessible sur PC (application), smartphone/tablette (app iOS/Android) et navigateur (outlook.office365.com).

## OneDrive

Service de stockage personnel Microsoft. La version professionnelle (icône bleue, incluse dans MySerenity) synchronise le Bureau, Mes Documents et les Images avec le cloud Microsoft.

## SharePoint

Outil de partage collaboratif Microsoft remplaçant le serveur de fichiers local. Héberge les répertoires partagés de l'organisation dans le cloud, accessibles depuis PC, smartphone ou navigateur.

## Microsoft Teams

Hub collaboratif Microsoft centralisant messagerie instantanée, visioconférence, partage de fichiers, équipes et canaux thématiques.

## OneNote

Bloc-notes numérique collaboratif. Prise de notes texte, audio, dessin, photo. Synchronisation instantanée entre appareils.

## Microsoft To Do

Gestionnaire de tâches simple intégré à Outlook. Synchronisé sur tous les appareils via Exchange.

## Microsoft Planner

Outil de planification en mode Kanban. Gestion des tâches par tableau visuel avec attribution et suivi d'avancement.

## Microsoft Stream

Plateforme de partage vidéo interne à l'organisation. Enregistrement d'écran, sous-titres automatiques, gestion des droits d'accès.

## Microsoft Bookings

Portail de prise de rendez-vous en ligne connecté à l'agenda Outlook. Permet à des personnes extérieures de réserver des créneaux disponibles.

## MFA (Multi-Factor Authentication)

Double authentification : second facteur de vérification en plus du mot de passe. Code SMS ou validation via Microsoft Authenticator. Imposé par Microsoft pour tout nouveau périphérique ou navigateur.

## MSP (Managed Service Provider)

Prestataire informatique gérant, supervisant et maintenant le système informatique d'un client en mode forfait illimité et proactif.

## Navigation privée

Mode de navigation ne mémorisant ni identifiants, ni cookies, ni historique. À utiliser lors d'une connexion depuis un poste partagé ou invité.
