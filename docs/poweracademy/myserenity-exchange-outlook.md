# Exchange & Outlook

Exchange est le service de messagerie professionnelle Microsoft hébergé dans le cloud. Outlook est le client utilisé pour le gérer depuis un PC, un smartphone ou un navigateur.

## Exchange — ce que vous avez

- 50 Go de boîte e-mail (extensible à 100 Go)
- Synchronisation en temps réel (mode push) sur tous les périphériques
- Agendas, carnets de contacts, gestionnaire de tâches
- Boîtes partagées possibles (contact@, compta@, fournisseur@...)

## Outlook — accès

| Support | Accès |
|---|---|
| PC | Application Outlook |
| Smartphone / Tablette | Application Outlook iOS / Android |
| Navigateur | [outlook.office365.com](https://outlook.office365.com) |

!!! tip "Poste en panne"
    Accéder à Outlook via le navigateur sur n'importe quel poste — aucune donnée n'est perdue.

## Procédures utiles

### Ne pas marquer les e-mails comme lus automatiquement

1. Ouvrir Outlook → Fichier → Options
2. Sélectionner "Courrier" → "Volet de lecture"
3. Décocher "Marquer l'élément comme lu lors du changement de sélection"
4. Valider avec OK

### Créer un dossier de recherche

1. Rubrique e-mail → Dossier → "Nouveau dossier de recherche"
2. Choisir l'adresse e-mail cible
3. Nommer le dossier et définir les critères (contact, non lus, prioritaires...)
4. Valider avec OK

### Rattraper un e-mail envoyé par erreur

1. Aller dans "Éléments envoyés"
2. Double-cliquer sur l'e-mail → Fichier → "Renvoi ou Rappel"
3. Sélectionner "Rappeler ce message" ou "Remplacer par un nouveau message"

!!! warning "Limite de cette fonction"
    Le rappel n'est pas garanti à 100 %. Il fonctionne mieux en interne si l'e-mail n'a pas encore été lu.

### Partager son calendrier

1. Rubrique Calendrier → sélectionner son calendrier → "Autorisations de partage"
2. Configurer les droits pour "Mon organisation" (occupé / fonctions / tous les détails)
3. Optionnel : ajouter des droits spécifiques à un collaborateur

### Créer une réunion Teams depuis Outlook

1. Créer un nouveau rendez-vous dans l'agenda
2. Ajouter les participants et nommer la réunion
3. Cliquer sur "Réunion Teams" pour générer le lien
4. Envoyer l'invitation
